# InventoryManagementSystem

## Overview

InventoryManagementSystem is a C# Windows Forms desktop application developed as part of my Software Engineering degree. The application allows users to manage parts and products through a structured inventory system while enforcing validation rules and business logic.

This project demonstrates object-oriented programming principles, validation logic, and CRUD functionality within a desktop application.

## Features

### Parts Management
• Add new parts  
• Modify existing parts  
• Delete parts  
• Search parts  

### Product Management
• Add new products  
• Associate parts with products  
• Remove associated parts  
• Prevent deletion of products with associated parts  
• Modify existing products  

### Validation
• Inventory must be between minimum and maximum values  
• Minimum must be less than maximum  
• Products must contain at least one associated part  
• Error messages for invalid operations  

## Technologies Used

• C#  
• .NET Windows Forms  
• Object-Oriented Programming  
• Visual Studio  
• DataGridView controls  

## Concepts Demonstrated

• Inheritance (InHousePart vs OutsourcedPart)

• Data validation
